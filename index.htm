What's Flexy Tooltip
Flexy Tooltip is a powerfull,funny and flexible jQuery plugin that makes your pages and page elements more interesting with importing some contents in the tooltip of them. it's also so easy to use and gives properties to handling what you exactly want.
some of them is :
  • Content of tooltip : The content of tooltip may be plain text or html code.
  • Mode of tooltip : When tooltip does its work,when mouse hovers an element or when focus occures on element or both.
  • Effects of tooltip : How tooltip appears or disappears and how much should has speed.
  • Position of tooltip : Tooltip must places a fixed position or not and also specifying where it must to be.
  • Making delays : Making delays before appearing or disappearing of tooltip.
  • Making flashes : Making flashes with appearing or disappearing of tooltip and also specifying color,count and interval of flashes.
  • Firing of events : You may want to do some functions before or after of appearing,before or after of disappearing and after flashing of appearing or disappearing done.


How To Use
Using The Flexy Tooltip is as simple as 3 following steps :
Step #1      Refrence the scripts to your page :

<script src="js/jquery.js" type="text/javascript"></script>
<script src="js/jquery.flexytooltip.js" type="text/javascript"></script>


Step #2      Link your page to css file :

<link href="css/flexytooltip.css" rel="stylesheet"  type="text/css"  />


Step #3      Fire tooltip method for every element you want :

<script type="text/javascript">
$(document).ready(function () {
$('#img1').tooltip('Your Content');
});
</script>
Method Signature
This is the main signature for calling Flexy Tooltip :

jQuery   tooltip(string||number||element||jQuery content , object options)

This means :
  • This method returns jQuery object.
  • First argument is the content that you want to show in tooltip and can be string,number,element or jQuery data types.
  • Second argument is an object for configuring tooltip and below is list of properties for this object :


Mode of tooltip

Property Description Default Value
mouseMode Set true if tooltip must be always beside of mouse cursor,otherwise false. true
focusMode Set true if tooltip appears when element is focused and disappear when element is blured,otherwise false. false


Effects of tooltip

Property Description Default Value
appearEffect The effect of appearing.it can be 'fade','slide','show' or 'none'. 'fade'
appearSpeed The effect speed of appearing.it can be 'slow','normal','fast' or a number in millisecond unit. 'normal'
disappearEffect The effect of disappearing.it can be 'fade','slide','show' or 'none'. 'fade'
disappearSpeed The effect speed of disappearing.it can be 'slow','normal','fast' or a number in millisecond unit. 'normal'




Tooltip opacity

Property Description Default Value
opacity Set a number between 0 and 1 for tooltip opacity. 0.8


Position of tooltip

Property Description Default Value
positionIsFixed Set true if you want fixed tooltip,otherwise false. false
position An array for specifying position of tooltip.first index is for top offset and second is for left offset. [10,10]


Making delays

Property Description Default Value
appearDelay Set a number in millisecond unit for delay before apperaing. 0
disappearDelay Set a number in millisecond unit for delay before disapperaing. 0


Making flashes

Property Description Default Value
appearFlash Set true if you want tooltip appears with flashing,otherwise false. true
appearFlashColor Set the color of flashing.it can be any valid css color. 'White'
appearFlashInterval Set a number in millisecond unit to specifying how much long every flash. 100
appearFlashCount Set a number for specifying count of flash(es). 2
disappearFlash Set true if you want tooltip disappears with flashing,otherwise false. true
disappearFlashColor Set the color of flashing.it can be any valid css color. 'White'
disappearFlashInterval Set a number in millisecond unit to specifying how much long every flash. 100
disappearFlashCount Set a number for specifying count of flash(es). 2


Firing of events

Property Description Default Value
beforeAppear Set this to a function(handler) if you want to do some works before appearing starts. function () { }
appearFlashCompleted Set this to a function(handler) if you want to do some works when flashing of appearing completed. function () { }
appearCompleted Set this to a function(handler) if you want to do some works when appearing completed. function () { }
beforeDisappear Set this to a function(handler) if you want to do some works before disappearing starts. function () { }
disappearFlashCompleted Set this to a function(handler) if you want to do some works when flashing of disappearing completed. function () { }
disappearCompleted Set this to a function(handler) if you want to do some works when disappearing completed. function () { }


There is some notices that you must to know :

  • If positionIsFixed = false,the position of tooltip that is defined with position property,is relative to position of mouse cursor and if positionIsFixed = true,the position of tooltip is relative to position of caller element.
  • position property can be positive or negative values for its first or second index.
  • If focusMode = true,position of tooltip is always fixed although you set positionIsFixed = false.
  • If disappearSpeed < disappearFlashCount * disappearFlashInterval,means time of flashing of disapperaing is longer than time of disappering and it's unappealing.so in this case disappearFlashCount is kept its value but disappearFlashInterval is changed automatically so that flashing and disappearing complete together.
  • CSS Styles can be apply or change to tooltip by editing 'flexytooltip.css' file.
